	# Using linux as a daily driver
	Ncommander, Action Retro, Adrian Black, Tech Time Traveller, Usagi Electric, and Jan Beta. These are youtubers who work on, show off, or enjoy vintage computers. I am not them but I'm a fan of all of them. My goals align with Ncommander's more than the rest, but their video styles all inspired this writing. I've been running Linux, a unix like, since 2012, and I've been a fan of unix and unix-likes for a long time. I recently used OpenBSD on my Raspberry Pi RPi400 keyboard computer. I decided against using it as a daily driver because I got annoyed with the lack of EFL software. I should have looked up how to install it on openBSD but, alas, I didn't. I run my RPi400 as my daily driver, so I'm well aware of and am perfectly comfortable with the Jank that comes with that. I have manually installed ee, aee, and xae, all of which are fairly nice to use. I wrote this entire thing in aee. I'll link to the repositories in the footnotes. My desktop is FVWM with "MyFVWM-EXT" which is nice. I wrote a password generator in bash. it's neat. I did try using Fedora with EFI but the EFI got corrupted somehow. IDK how. Probably because it was fat16. Regardless, I'm currently running ubuntu on it. It may be necessary for me to edit a config to fix the ram usage.
